configure (
| substation:Configuration config ,
) {

Set configuration options for substation as a monad. You may call configure multiple times to assemble configuration options - in case of conflict, the most recent configuration wins.

You may still instantiate substation instances after configuring and starting the monad. They will not inherit configuration options or Actions from the monad.

Arguments

| substation:Configuration config
