Realtime application gateway and authentication provider. This is the all-purpose module for writing client applications over the submergence layer. You may serve an application from a local cluster or respond to events forwarded from a submergence service. To access the client library, bundle this module with browserify.

A substation always represents a single domain (or subdomain) which must appear in the configuration.
