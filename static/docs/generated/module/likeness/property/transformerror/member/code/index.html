member String | :likeness:TransformError#code

A String Error code intended to remain stable and reliable. If you need to branch on the type of Error that occured, this is your String. Possible values are:

  • MISSING A key was missing, or a .exists constraint failed.
  • INVALID A ValidationError was generated by the input \ document or post-transform validation of the target.
  • FORMAT Failed to parse a String for .cast or .group.
  • ILLEGAL Unexpected extra keys, keys rejected by .key.
  • LIMIT Transform violates length constraints.
  • SYNC An asynchronous .transform constraint could not be processed.
