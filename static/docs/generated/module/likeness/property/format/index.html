module :likeness:Format

Perform Web-RFC string format validations. Supports the following formats:

  • "date-time" rfc-3339 timestamps.
  • "date"
  • "time"
  • "email" Supports the most commonly used parts of rfc-5322.
  • "hostname" rfc-1034 internet host names.
  • "ipv4" rfc-2673 IPv4 addresses.
  • "ipv6" rfc-2373 IPv6 addresses.
  • "uri" rfc-3986 universal internet resource identifier.

Source borrows heavily from another validating lib.
