Create a new document representing the data from one provided document accumulated into another, using transform constraints defined by this Likeness. Honors validation constraints on the result document (and not on the input document) to ensure that only validating documents are produced.

Arguments

target

Original/accumulated document onto which the transform is applied to create a result document.
value

Source document to merge into target.
optional
callback (
| Error | undefined err ,
newValue ,
) {

If present, the callback will always be honored. However, it is only necessary if this likeness has an asynchronous transform operation.
Arguments
| Error | undefined err

If the transform failed because of issues with the input or result documents, an error report is passed.
newValue

The new value produced by this Likeness' transform constraints.

Return Values

If no callback was provided (and this Likeness is not asynchronous) returns the transformed value.
