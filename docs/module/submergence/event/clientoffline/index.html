event :submergence+clientOffline

When the last live connection for a client id goes offline, exactly one substation instance in your cluster will emit this event once. It is emitted for every client belonging to a user.

Arguments

| String domain

The internet hostname where this User is logged in.
| String user

The User ID that has just appeared offline.
| String client

The Client ID that has just appeard offline.
